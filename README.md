# CPTS427-EA2

For my activity I made a python script that anaylzes standard Apache access logs, summarizes web traffic, and detects suspicious activity at a set threshold.  

It takes a log file as input and outputs:  
- Top IP addresses by number of requests
- Number of unique IPs
- Suspicious IP detection based on high traffic, repeated errors, or scanning many pages
- Top requested pages
- HTTP code summary
- Overall error rates
